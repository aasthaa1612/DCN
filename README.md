# Data Communication and Networking (DCN) Lab

Welcome to the **Data Communication and Networking (DCN)** repository. This repository contains a collection of lab experiments, network designs, and routing configurations created using **Cisco Packet Tracer** and **Python**.

## 📌 Repository Overview

This repository is dedicated to practical implementations of networking concepts, routing protocols, and topology designs. It includes configurations for:
- Static Routing
- Dynamic Routing (RIP, OSPF)
- LAN and WAN Topologies
- Network Layer Protocols and Subnetting

---

## 📂 Key Files & Configurations

| File / Component | Description |
| :--- | :--- |
| `7.2 (1).pkt` | Subnetting and basic IP addressing configuration layout. |
| `class1.pkt` | Basic network topology with switches and routers. |
| `exp static.pkt` | Static routing simulation between multiple networks. |
| `exp5.pkt` | Lab Experiment 5: Configuration of basic routing tables. |
| `exp7.pkt` | Lab Experiment 7: Implementation of Local Area Networks (LAN). |
| `exp9.pkt` | Lab Experiment 9: Network services configuration (DNS, DHCP, HTTP). |
| `exp_dynamic.pkt` | Dynamic routing configuration (e.g., OSPF / RIPv2). |

---

## 🛠️ Prerequisites

To run and view the files in this repository, you will need:
1. **Cisco Packet Tracer** (v8.0 or higher recommended) to open and simulate `.pkt` files.
2. **Python 3.x** for executing `.py` network script utilities.

---

## 🚀 How to Use

1. **Simulate a Topology**:
   - Open **Cisco Packet Tracer**.
   - Go to `File -> Open` and select any of the `.pkt` files.
   - Run the simulation or use the real-time mode to test connectivity using PING or Web Browser tools within the simulation.

2. **Run Python Scripts**:
   - Execute the script using Python:
     ```bash
     python expt3-2.py
     ```
